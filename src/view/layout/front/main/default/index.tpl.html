功能介绍

提供了上行信息处理,可以轻松开发出各种基于信息交互的应用

文章系统自动携带广告功能，一旦设置应用的广告，每一篇分享的文章在生成时都会自动带上广告，助理产品运营，简化发布流程

常见问题

  • 问题1：基于框架可以做什么？
  • 小熊猫微信开发框架本身集成了微信公众平台消息上行处理，通过简单的继承默认的处理类实现对应消息格式的回复就可以完成各种各样的智能机器人。另外，框架本身实现了文章管理系统，且文章系统完全适配web和移动端浏览。
  • 问题2：依赖的环境有哪些？
  • 框架完全基于SAE开发，通过SAE应用仓库可以快速安装。
